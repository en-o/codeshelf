◈ VISUAL_ELEMENTS
  • 六边形网格底层：象征数据结构与技术架构，半透明营造全息感
  • 三层发光层板：代表"书架"核心概念，渐变发光模拟全息投影

◈ TECHNICAL_FEATURES
  • SVG 原生 SMIL 动画：呼吸灯、流动线、扫描线，导出 PNG 时自动定格

  • 故障闪现层：随机出现的粉色横条，赛博朋克标志性视觉
  • 玻璃质感基底：半透明深蓝黑圆形，适配深色模式 Dock

◈ PROJECT_CONCEPT
优化后的全息图标深度融合了 CodeShelf 的三大核心：
1. 书架整理：三层发光层板直观展示项目层级
2. Git 管理：顶部悬浮节点可视化多远程仓库同步
3. 代码本质：中层代码块与底层命令行强化开发者属性
◈ Tauri_INTEGRATION
建议配置：
• macOS：保留完整发光滤镜，禁用 template 模式以保持霓虹色
• Windows：导出 ICO 时包含 16/32/48/256 多尺寸，16px 自动简化
• Linux：适配深色主题，图标基底透明度确保与面板融合
◈ ANIMATION_NOTE
SVG 内含 SMIL 动画（呼吸灯、流动线），当转换为 PNG 时会自动定格。
如需在 Tauri 启动界面使用动画，可直接使用此 SVG，或导出序列帧制作 loading 动画。
